# Time Series Analysis using Machine Learning  
### Exercises & Implementations in Python

This repository contains my work for the **[Time Series Analysis using Machine Learning](https://opiskelijanopas.tuni.fi/en/tampere-university/curriculum/course-units/otm-f6c89a40-e9c7-4f01-bf54-51f2b2ef2d70?year=2025)** course.  
It includes Python-based exercises covering preprocessing, noise reduction, feature extraction, classification, prediction, and time–frequency analysis.

---

## Course Topics Covered

- Time series preprocessing & noise suppression  
- Filtering, outlier handling, and data cleaning  
- Correlation and dependency analysis  
- Feature extraction from time series  
- Linear prediction models (AR, regression)  
- Machine learning–based prediction (RNN family)  
- Time–frequency domain understanding  
- Model selection and evaluation  
- Practical classification and forecasting tasks
